# Writing Effective Learning Objectives

A portfolio-quality microlearning experience (~5–7 minutes) for new instructional designers, trainers, and subject matter experts.

**Live demo:** [jacobjones.dev](https://jacobjones.dev) *(or deploy via GitHub Pages — see below)*

---

## What's Inside

| File | Purpose |
|---|---|
| `index.html` | Course structure, all 12 slide sections |
| `styles.css` | Full design system — tokens, layout, animations |
| `main.js` | Course logic, activity scoring, slide navigation |

## Learning Objectives Covered

- Differentiate effective and ineffective learning objectives
- Apply Bloom's Taxonomy to select measurable action verbs
- Write learning objectives aligned with desired learning outcomes

## Course Structure

| Slide | Content Type |
|---|---|
| 1 | Introduction — why objectives matter |
| 2 | Anatomy of an effective objective |
| 3 | Bloom's Taxonomy overview |
| 4 | Strong vs. weak examples (side-by-side) |
| 5 | **Activity 1:** Spot the weak objective |
| 6 | **Activity 2:** Pick the right Bloom's verb |
| 7 | Rewrite prep — tips and formula review |
| 8 | **Activity 3:** Rewrite two ineffective objectives |
| 9–11 | **Knowledge Check** (3 questions with feedback) |
| 12 | Completion screen with score and key takeaways |

## Deploying to GitHub Pages

1. Create a new GitHub repository
2. Upload `index.html`, `styles.css`, and `main.js` to the root
3. Go to **Settings → Pages**
4. Set Source to `main` branch, root folder
5. Your course will be live at `https://yourusername.github.io/repo-name`

## Design Notes

- **Palette:** Deep navy `#0F1923` · Amber `#E8A838` · Slate card `#1C2E3F`
- **Typography:** DM Serif Display (headings) + Inter (body)
- **Signature element:** Milestone progress track with animated nodes
- No frameworks or build tools — pure HTML, CSS, and JavaScript
- Keyboard-navigable (arrow keys)
- Respects `prefers-reduced-motion`
- Responsive down to mobile (380px+)
